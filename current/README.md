# TAIFEX BOT V5.1.2

台指期多商品多策略模擬交易機器人。

## V5.1.2 更新

- 監測行情與機器人交易拆成兩個按鈕控制
- 達每日最大交易次數時，只暫停機器人交易，監測行情繼續
- 達每日最大虧損時，若有持倉會先平倉，再暫停機器人交易，監測行情繼續
- 移除平倉冷卻，避免妨礙停損、停利、強制平倉等風控
- 保留開倉冷卻
- 新增停損後重新進場冷卻秒數
- 新增交易統計面板
- 新增 settings.json 自動儲存/讀取
- 匯出交易紀錄檔名為 trades_v5_1_2.csv

## 執行

```bash
pip3 install --break-system-packages -r requirements.txt
python3 main_gui.py
```
